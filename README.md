# Слот машина
Описание: Данное приложение моделирует поведение слот машины. Игрок вращает барабан и на поле 3x3 (будем дальше
называть это сеткой)
выпадают различные объекты (символы, бонусы, призы). Объекты будут описаны далее. После выпадения могут появиться
различные комбинации, которые инициируют разные события, события будут описаны далее. Игрок платит деньги и
может получить выигрыш в зависимости от своей ставки. Также в ходе игры могут активироваться разные режимы, о режимах
будет описано далее.
Данное приложение эмулирует работу слот машины, находящуюся по адресу: https://betsoft.com/play-demo/?id=5707.
Там можно с UI проверить работу, там же можно подробнее почитать о правилах

## Объекты:
### Символы
Это самые обычные объекты в слот машине. Если они попадают три в ряд, то игрок получает выигрыш. 
Комбинация три в ряд может быть составлена на любой горизонтальной, вертикальной или диагональной линии
(главные диагонали сетки). Каждый символ дает разное вознограждение (зависит от размера ставки)
и имеет разную вероятность выпадения. Также 
есть символ WILD, который помимо составления комбинаций с себе подобными символами, составляет комбинации с любыми 
другими. Например, комбинация 3 в ряд: WILD, WILD, JUG. Даст выигрыш соразмерный комбинации JUG JUG JUG. 
Таблица выигрышей  

| Символ        | Выплата за 3 в ряд при ставке 1 денег |
|---------------|---------------------------------------|
| GREEN_STONE   | 0.2                                   |
| PURPLE_STONE  | 0.4                                   |
| ORANGE_STONE  | 0.6                                   |
| RED_STONE     | 0.1                                   |
| JUG           | 1.6                                   |
| EYE           | 2.0                                   |
| PYRAMID       | 3.0                                   |
| WINGS         | 5.0                                   |
| WILD          | 10.0                                  |

### Призы
Призы выпадают во всех режимах, но призы играют роль только в режиме **HOLD&WIN**. В режиме **HOLD&WIN** 
фиксируются на сетке бонусы RaBonus (при всех последующих вращениях барабана RaBonus остается на месте, до конца
режима **HOLD&WIN**), и любой выпавший приз имеет номинал награды, эта награда аккумулируется в RaBonus-е. Например,
в центре сетки зафиксирован бонус Ра, при вращении барабаны выпадывают призы, каждый со своим номиналом, эти награды
аккумулируются в бонусе Ра находящемся в центре. По окончанию режима, они превращаются в выигрыш. Таблицы номиналов
призов приведена далее (не зависит от ставки)

| Приз    | Награда |
|---------|---------|
| GRAND   | 300     |
| MAJOR   | 45      |
| MINOR   | 15      |
| MINI    | 7.5     |

### Бонусы
Есть 2 вида бонусов. Обычный и Ра бонус. Бонусы могут выпасть только в 1 и 3 вертикале сетки. В то время как 
бонус Ра может выпасть только на серединной вертикале. После вращения барабана может быть несколько сценариев:
1. Выпал 1 любой бонус  
В таком случае срабатывает механика **CHEST OF GOLD** и с определенным шансом игра может перейти в режим **HOLD&WIN**.
2. Выпало любое количество обычных бонусов  
В таком случае срабатывает механика **CHEST OF GOLD** и с определенным шансом игра может перейти в режим **HOLD&WIN**.
Чем больше бонусов тем выше шанс.
3. Выпал 1 Ра бонус и 1 обычный
Игра в случайном месте сетки ставит бонус. И если получится случай как в пункте 4, то активируется режим
**HOLD&WIN**
4. Выпал 1 Ра бонус и больше 2 обычных
Игра гарантированно переходит в режим **HOLD&WIN**

Также все обычные бонусы имеют свой номинал в размере от 0.3 до 4.5 (выбирается случайным образом при создании)
и в режиме **HOLD&WIN** эти награды также собираются в Ра бонус.

## Режимы
### Обычный
В ходе этого режима игрок делает ставку крутит барабан и ему выпадают комбинации. Есть всего два возможных сценария:
1. Выпала комбинация из символов
Тогда игрок просто получает выигрыш
2. Бонусы описанные в предыдущем разделе составили свою комбинацию и режим переключился в **HOLD&WIN**

### HOLD&WIN
При переходе в данный режим игрок автоматически получает 3 бесплатных прокрутки барабана. Комбинации из символов
больше не играют роли. Но зато теперь все призы выпавшие в сетке и обычные бонусы отдают свою награду бонусам Ра,
находящимся на сетке. Награда аккумулирется в них и по окончанию режима уходит к игроку. Все Ра бонусы бывшие до запуска
режима замораживаются и остаются на сетке. Каждый раз когда выпадывает новый Ра бонус, то количество вращений барабанов
возвращается к 3.

### Инструкция

1. Способ запуска через Intellij Idea
Есть тестовый класс Main. Достаточно запустить его, ввести количество итераций
чтобы приложение выдало статистику (ставка фиксированна и равна 5).  
Пример вывода:  
Total Bet: 500  
Total Win: 343  
Total Hit Rate: 43.00%  
Return to Player (RTP): 68.60%  

2. Способ запуска через jar файл (как по заданию)
    1. Находясь в корне проекта выполнить команду mvn clean package, чтобы собрать jar файл
   2. Далее ввести команду java -jar target/math-test-introduction-1.0-SNAPSHOT.jar
   3. Программа запущена

3. Третий способ активирует секретный режим (это я от себя добавил)
   1. Находясь в корне проекта выполнить команду mvn clean package, чтобы собрать jar файл
   2. Далее ввести команду java -jar target/math-test-introduction-1.0-SNAPSHOT.jar cheat_code
   3. Программа будет запущена не в режиме сбора статистики, а в режиме игры. Можно будет поиграть самому :)
